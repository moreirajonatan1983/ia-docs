# Monitor and Track

> **Source:** [kiro.dev/docs/enterprise/monitor-and-track/](https://kiro.dev/docs/enterprise/monitor-and-track/)

---

El monitoreo es una parte importante del mantenimiento de la confiabilidad, disponibilidad y rendimiento de Kiro para tu organización.

---

## Kiro Built-in Monitoring

| Feature | Descripción | Más info |
|---|---|---|
| **Dashboard** | Métricas agregadas de actividad de usuarios suscriptos | [Viewing Kiro usage on the dashboard →](https://kiro.dev/docs/enterprise/monitor-and-track/dashboard) |
| **User activity reports** | Actividad individual de cada usuario en Kiro | [Viewing per-user activity →](https://kiro.dev/docs/enterprise/monitor-and-track/user-activity) |
| **Prompt logs** | Registro de todos los prompts que los usuarios ingresan en el chat de Kiro | [Logging users' prompts →](https://kiro.dev/docs/enterprise/monitor-and-track/prompt-logging) |

---

## AWS Monitoring Tools

| Tool | Uso |
|---|---|
| **AWS CloudTrail** | Captura llamadas a la API y eventos relacionados; registra qué usuarios llamaron a AWS, desde qué IP y cuándo |
| **Amazon CloudWatch** | Monitorea recursos AWS en tiempo real; creá dashboards personalizados, alarmas y rastreá métricas como número de invocaciones de Kiro o usuarios activos diarios |
