# Learn by Playing

> **Source:** [kiro.dev/docs/guides/learn-by-playing/](https://kiro.dev/docs/guides/learn-by-playing/)

---

Kiro ofrece un recorrido interactivo de aprendizaje donde podés aprender AI Engineering completando tareas reales dentro de un videojuego. No solo podés personalizar el juego — también podés personalizar Kiro extendiendo su contexto y comportamientos con Model Context Protocol (MCP).

---

## Learning Path

El tutorial está estructurado en etapas progresivas:

| Etapa | Contenido |
|---|---|
| **00 - Setup** | Configurar el entorno de desarrollo y lanzar el juego |
| **01–06 - Tasks** | Completar tareas dentro del juego usando las capacidades de Kiro |
| **07 - Extending Kiro with MCP** | Extender Kiro con servidores MCP para capacidades adicionales |
| **99 - Conclusion** | Cierre del recorrido y próximos pasos |

---

## Getting Started

1. Ir a [kiro.dev/docs/guides/learn-by-playing/00-setup/](https://kiro.dev/docs/guides/learn-by-playing/00-setup/)
2. Seguir las instrucciones para configurar el entorno de desarrollo
3. Lanzar el juego y comenzar las tareas guiadas
4. A medida que completás las tareas, aprendés las funcionalidades clave de Kiro

---

## What You'll Learn

- Capacidades del chat y modos de trabajo (Vibe vs Spec)
- Creación y gestión de Hooks para automatizar el workflow
- Configuración de MCP servers para extender las capacidades
- Uso de Steering para guiar el comportamiento del agente
- Integración de Kiro en un flujo de desarrollo real
