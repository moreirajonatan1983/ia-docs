# Governance

> **Source:** [kiro.dev/docs/enterprise/governance/](https://kiro.dev/docs/enterprise/governance/)

---

Como administrador, podés controlar qué modelos y MCP servers están disponibles para tus usuarios. Los controles de governance se gestionan desde la **Kiro console → Settings → Shared settings**.

---

## Model Governance

Por defecto, los usuarios pueden acceder a cualquier modelo soportado por Kiro.

**Opciones de control:**
- **Activar gestión de acceso a modelos** → seleccioná una lista aprobada de modelos
- **Establecer un modelo por defecto** → se aplica automáticamente a todos los clientes

Para detalles: [kiro.dev/docs/enterprise/governance/model](https://kiro.dev/docs/enterprise/governance/model)

---

## MCP Governance

Por defecto, los usuarios pueden usar cualquier MCP server en su cliente Kiro.

**Opciones de control:**
- **Deshabilitar MCP completamente** para todos los usuarios del profile
- **Especificar una allow-list** de MCP servers validados a través de un MCP registry

Estas políticas pueden configurarse a nivel de organización o sobreescribirse por cuenta.

Para detalles: [kiro.dev/docs/enterprise/governance/mcp](https://kiro.dev/docs/enterprise/governance/mcp)
