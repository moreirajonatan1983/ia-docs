# Code References

> **Source:** [kiro.dev/docs/privacy-and-security/code-references/](https://kiro.dev/docs/privacy-and-security/code-references/)

---

Las **code references** son atribuciones que aparecen cuando el código generado por Kiro es similar a código open-source disponible públicamente. Esto te permite revisar el origen y tomar decisiones informadas sobre su uso.

---

## View the Code Reference Log

El log de code references muestra las sugerencias de código similares a código público.

1. Abrí la pestaña **Output** en la barra de estado
2. Desde el menú dropdown, elegí **code-references**

> ℹ️ El log solo aparece cuando code references está habilitado y Kiro generó código con referencias.

---

## Turn Code References On and Off

Code references están **deshabilitadas por defecto**.

Para activarlas o desactivarlas:
1. Abrí **Settings** en Kiro
2. Cambiá a la sub-pestaña **User**
3. Elegí **Kiro** (o buscá "code references" en el search bar de Settings)
4. Bajo **Code References: Reference Tracker**, marcá o desmarcá la opción **Allow Kiro to generate code with references**

---

## Opt Out as an Enterprise Administrator

Los administradores Enterprise pueden deshabilitar globalmente las sugerencias de código con referencias para **todos los usuarios** del profile.

- Los usuarios Enterprise no tienen control sobre este setting — está controlado por el administrador y no puede sobreescribirse
- Se configura desde la **Kiro console** → Settings

Para más información: [Enterprise settings →](../13_Enterprise/08_Settings.md)
